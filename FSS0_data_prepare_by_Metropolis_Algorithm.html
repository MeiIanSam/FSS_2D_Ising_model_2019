We are going to simulate the 2D Ising Model with square lattice and PBC, here is some functions I define: the initial state, the neigbors of a certain site, and the energy of the system.

The Hamiltonian of Ising Model is $H= -J\sum_{<i,j>}\sigma_{i}\sigma_{j} + h\sum_{i}\sigma_{i}$,note that here we assume $h=0, J=1$

def initsys(L):
    arr_init=np.random.randint(2,size=(L,L))
    for i in range(L):
        for j in range(L):
            if arr_init[i][j]<1:
                arr_init[i][j] = -1
    return arr_init

def neighbors(arr,i,j,L):
    neighbordic = [(i,(j+1)%L),(i,j-1),((i+1)%L,j),(i-1,j)]
    return neighbordic
#neighbor=neighbors(arr,3,3,8)

def siteE(arr,i,j,L):
    E=0
    J=-1
    for neighbor in neighbors(arr,i,j,L):
        E+=J*(arr[neighbor]*arr[i][j])
    return E

def totalE(arr,L):
    E=0
    for i in range(L):
        for j in range(L):
            E+=siteE(arr,i,j,L)/2
    return E

Here is the Metropolis algorithm.

The Metropolis algorithm is based on the entropy maximizing principle and the detail balance principle, we can know the how the system change during the temperature fluctuation, then we know that is the competition between energy and temperature (concept of free energy).

The probability of changing confriguation is $P= e^{\frac{\Delta E}{k_{b}T}}$ if $\Delta E > 0$ (thermal equalibrium); else $P=1$.

then we know, for a certain temperature T, we can simulate its equalibrium confriguation by MC steps.

#trail all
def Metropoly(arr,steps,L,T,sequential):
    kb=1
    beta=1/(kb*T)
    Msum=[]
    M2sum=[]
    Esum=[]
    E2sum=[]
    E=totalE(arr,L)
    xs=[i for i in range(L)]
    ys=[j for j in range(L)]
    if sequential==False:
        random.shuffle(xs)
        random.shuffle(ys)
    for step in range(steps):
        for (x,y) in itertools.product(xs,ys):
            dE=-2*siteE(arr,x,y,L)
            W=math.exp(-dE*beta)
            accprob=min(1,W)
            randomnumb=np.random.random()
            if accprob>randomnumb:
                arr[x][y] = -arr[x][y]
        M=np.sum(arr)
        E=totalE(arr,L)
        Msum.append(abs(M))
        M2sum.append(M**2/10**5)
        Esum.append(E)
        E2sum.append(E**2)
    Mavg=sum(Msum)/steps
    M2avg=sum(M2sum)*10**5/steps
    Eavg=sum(Esum)/steps
    E2avg=sum(E2sum)/steps
    chi=(M2avg-Mavg**2)*beta
    chi=chi/(L**2)
    C=(E2avg-Eavg**2)*beta/T
    C=C/(L**2)
    Mavg=Mavg/(L**2)
    Eavg=Eavg/(L**2)
    return Mavg,Eavg,chi,C
